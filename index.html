Я начинаю новый цикл изучения веб-кодинга. Сегодня мы начнем изучать фреймворк Bootstrap 5!

Сразуначнем разбирать, и начнем мы с текстовых классов. Ну и немного новых тегов

Для начала теги:

  • Mark- для Выделения текста

  • Del- для зачеркивания куска текста

  • U- для подчеркивания текста

  • small- для мелкого текста(здесь не сильно видно, но видно)

  • strong - для жирных, в плане текста

  • em - для курсива

  • abbr- Делает подсказку при наведении
